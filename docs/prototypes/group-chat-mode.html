夜雨贴着旧城墙往下淌。他停在审判庭门前，听见火盆里传来低低的爆响。门后有人念出契约神的名字，像把一枚冷针慢慢推入他的耳中。
本轮讨论
这段是不是缺一个转折？
设定师
我觉得转折应该来自诅咒规则本身。不要只是外部审判升级，而是让规则第一次伤到主角真正珍视的东西。
剧情师
可以让妹妹的记忆第一次被夺走。这样冲突会从“他被审判”转向“他正在失去自己为什么反抗”。
审稿
这个转折有效，但要提前埋一个“声音”的伏笔，否则读者会觉得突然。
@写手 按这个继续写
写手

他忽然发现，自己想不起妹妹的声音了。那个曾经在雨夜里喊他回家的声音，像被谁从脑中轻轻抽走，只剩一片没有回响的空白。

审稿
这一句好，但前面需要多铺半拍，让“想不起声音”的瞬间更刺。
