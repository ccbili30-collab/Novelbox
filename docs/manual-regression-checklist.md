# TBird 手工回归清单

## 聊天主流程
- 新建会话
- 切换历史会话
- 发送用户消息
- 流式输出
- 停止生成

## 消息编辑流程
- 编辑用户消息并保存
- 编辑用户消息并重新发送
- 编辑 AI 输出并保存
- 编辑 AI 输出并继续
- 重新生成 AI 输出
- 删除消息并保留分支

## 小说资料流程
- 保存小说资料
- 导入正文 TXT
- 导出正文 TXT
- 同步 AI 输出到正文库
- AI 填充剧情线/角色卡/世界观/大纲/伏笔线

## 设置与上下文
- 修改 Base URL / API Key / 模型
- 拉取模型列表
- 调整上下文条数与无限上下文
- 打开上下文预览
- 修改排版参数与预设

## Android 宿主流程
- Android bridge 普通请求
- Android bridge 流式请求
- 文件选择
- 后台通知
