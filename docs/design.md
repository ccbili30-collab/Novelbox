# Roundtabox Design Direction

Roundtabox is a calm, mobile-first writing workspace for AI-assisted fiction.

The public design goal is simple:

- Keep the manuscript visible like a sheet of paper.
- Keep conversation compact and text-hugging.
- Let the user move naturally between one-on-one writing and a roundtable-style discussion.
- Avoid dashboard clutter while still leaving room for advanced creative workflows.

## Public Demo Scope

This repository demonstrates the product surface and safe client-side shell. The real roundtable engine, private memory logic, and private creator templates are not part of this public source tree.

## UI Principles

- The first screen should be usable, not a landing page.
- The manuscript area should feel stable and readable.
- Chat bubbles should hug the text.
- Controls should be compact, obvious, and close to the writing action.
- Motion should clarify state changes without making the whole screen jump.
