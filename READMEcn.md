# 圆桌小说盒子公开壳

圆桌小说盒子 Roundtabox 是一个移动端优先的 AI 小说创作工作区，核心方向是“多创作者圆桌共创”。

这个公开仓库只保留产品展示壳和非敏感客户端代码，用来展示方向、界面、普通写作体验、安卓壳、API 请求工具、会话基础结构和设计文档。

私有实现不包含在这里。

## 公开内容

- 移动端写作 UI 壳
- 普通聊天和正文小窗演示
- 会话、设置、工具函数、API 请求、安卓 WebView 壳
- 圆桌写作方向的公开设计说明

## 不公开内容

- 圆桌调度核心
- 私有记忆调度和创作者身份图谱
- 私有创作者模板和私有提示词
- 私有完整 APK
- 任何私有提示词或模型行为规则

## 一句话

圆桌小说盒子解决的是：让小说作者不再只和一个 AI 对话，而是能让多个带记忆、带立场的 AI 创作者围绕同一部作品协作讨论。

## 运行

```bash
node dev-server.mjs
```

打开终端输出的本地地址即可。

## APK 说明

可以打包公开壳 APK，但不要把完整私有版 APK 上传到公开仓库。APK 可以被解包，客户端里的 JS 和提示词都有暴露风险。
